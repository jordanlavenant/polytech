# Primitives

## Fonctions usuelles

| Fonction | Primitive |
| - | - |
| $x^n$ | $\frac{x^{n+1}}{n+1}$ |
| $\frac{1}{x}$ | $\ln(x)$ |
| $\frac{1}{x - a}$ | $\ln(x - a)$ |
| $e^x$ | $e^x$ |
| $e^{ax}$ | $\frac{e^{ax}}{a}$ |
| $\sin(x)$ | $-\cos(x)$ |
| $\sin(ax)$ | $\frac{-\cos(ax)}{a}$ |
| $\cos(x)$ | $\sin(x)$ |
| $\cos(ax)$ | $\frac{\sin(ax)}{a}$ |
| $\tan(x)$ | $-\ln(\cos(x))$ |
| $\frac{1}{\cos²(x)}$ | $\tan(x)$ |
| $\frac{1}{\sin²(x)}$ | $\frac{-1}{\tan(x)}$ |
| $\frac{1}{1+x²}$ | $\arctan(x)$ |

## Primitives de fonctions composées

| Fonction | Primitive |
| - | - |
| $\frac{u'}{u}$ | $\ln(u)$ |
| $\frac{u'}{1+u^2}$ | $\arctan(u)$ |
| $u'u^n$ | $\frac{u^{n+1}}{n+1}$ |
| $u'\cos(u)$ | $\sin(u)$ |
| $u'\sin(u)$ | $-\cos(u)$ |
| $\frac{u'}{u^2}$ | $\frac{-1}{u}$ |
| $\frac{u'}{\sqrt{u}}$ | $2\sqrt{u}$ |
